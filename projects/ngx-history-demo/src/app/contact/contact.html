‚ùì Frequently Asked Questions

How do I install NgxHistory?

Simply run npm install ngx-history in your Angular project. No additional setup required - the service uses providedIn: 'root' for automatic injection.

Which Angular versions are supported?

NgxHistory 2.x supports Angular 15+. For older Angular versions (11-14), use NgxHistory 1.x which is still maintained.

Does it work with Angular Router?

Yes! NgxHistory is designed to work seamlessly with Angular Router. It listens to routing events and maintains its own navigation stack.

Can I customize the history behavior?

The service provides a clean API for navigation control. While the core behavior is standardized, you can build custom logic on top of the observable streams.

Is there TypeScript support?

Absolutely! NgxHistory is written in TypeScript with full type definitions, interfaces, and IntelliSense support included.

We welcome contributions! Check out our GitHub repository for contribution guidelines, open issues, and development setup instructions.
