ğŸ¯ What is NgxHistory?

NgxHistory is a modern Angular library that provides browser-like navigation history functionality for single-page applications. It bridges the gap between traditional browser navigation and programmatic routing control.

While Angular Router provides excellent routing capabilities, it doesn't offer the same back/forward navigation experience that users expect from traditional web applications. NgxHistory fills this gap by tracking navigation state and providing intuitive controls.

  • Reactive State Management - Observable streams with RxJS integration

  • Modern Angular - Built with Angular 21 and latest patterns
  • Backward Compatibility - Works with existing Angular Router setup
  • Lightweight - Minimal overhead, tree-shakable design
  • Easy Integration - Simple service injection pattern
